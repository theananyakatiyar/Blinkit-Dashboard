
# Blinkit Sales Analysis Dashboard
An interactive Excel dashboard analysing Blinkit grocery sales data across various parameters.

## Overview
This dashboard provides insights into Blinkit's sales performance using dynamic charts, slicers, and KPIs.

## Key Metrics
- Total Sales: $1.2M
- Average Sales: $141
- Number of Items: 8523
- Average Rating: 3.97

## Features and Techniques Used
- Pivot Tables - to summarise sales, ratings, and item counts
- Pivot Charts - donuts, bar charts, and line chart built from pivot data
- Slicers - interactive filters for Outlet Size, Location Type, and Item Type
- Dynamic "View By" toggle - switching between Sales, Count, and Rating metrics

## Software 
- Microsoft Excel 

## Data Set 
- 8,523 records across multiple outlet types, locations and categories
- Fields include: Item Type, Fat Content, Outlet Size, Outlet Location, Sales, Rating, Item Visibility, Item Weight
